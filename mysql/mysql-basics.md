# MySQL Basics

### mysql.server
| Action | Command |
| ------ | ------- |
|Install with HomeBrew|`brew install mysql`|
|Start|`mysql.server start`|
|Stop|`mysql.server stop`|
|Command line client|<pre>mysql -u**user** -p**pass**</pre>|


### Queries

| Purpose | Query |
| ------ | ------- |
| List all databases | `SHOW DATABASES;` or `SHOW SCHEMA;`|
